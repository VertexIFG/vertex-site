# Vertex Infrastructure Group Site

Modern static React site for `vertexifg.com`.

## Stack

- Vite
- React
- TypeScript
- Playwright regression tests

## Local Development

```bash
npm install
npm run dev
```

## QA

```bash
npm run lint
npm run build
npm run test:e2e
```

## Production Build

```bash
npm run build
```

The production output is generated in `dist/` and can be deployed to any static host.
